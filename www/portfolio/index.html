Procedural dungeon (2024-2025)

Project using unity3D and POLYGON assets from Synty studio. Some assets were edited, and few created, using Blender (4.1)

This sandbox project aimed to be a dungeon generator system based on:

  • Tiles with flexible sizes
  • Tile content random generation
  • Specialized tiles: start, end, boss room
  • Dungeon main path and branches number random generation
  • Seed: to be able to share and regenerate a level (multi player)
  • Unity custom editors. For quickly configuring and debugging map builds.
  • Map biomes: tiles are organized by theme
  • Map consistency is checked on the fly

With this project I also dived into:

  • Synty Base locomotion and Sword combat systems
  • Network frameworks, looking for all topologies:
    • Unity netcode for game objects
    • Unity netcode for entities
    • Photon engine (Fusion2)

    RPG (2021-2023)

    Project using unity3D and POLYGON assets from Synty studio. Started in the holy 2020, I wanted to implement all the core mechanics that glue a game application, so far I have achieved:

    • Player with locomotion and animation system.
    • Character customization.
    • UI system, scaling with resolution and ratio.
    • Render pipeline, post processed effects, lights both GI & backed
    • Item managment:
      • Inventory system
      • Loot system
      • Db integration
    • The game save/load with world/player states.
    • Security around game saves and player at runtime.

    I plan to continue working on this project and will play with shadergraph for amazing effects or cinemachine because I love dolly cams and so much more...

    Nasty Bubbly (2012)

    This was quite a successful entry to Ludum Dare 21 for the 72H team JAM contest.

    Controls

    • UP and DOWN Arrows: Move the fish well, up and down.
    • SPACEBAR: Speed boost.

    The goal is to escape from the landing net to avoid being caught and consequently dying in excruciating suffering.

    Use the boost to go faster and pollute the bowl by doing so, avoiding obstacles such as bubbles and pillars to win the game. The fish food increases your boost bar.

    Enjoy! And for the love of nature, save this goldfish's life ! It's only a matter of Karma.
